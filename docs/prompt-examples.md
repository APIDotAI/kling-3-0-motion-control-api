## Production prompt notes

- Store prompt versions together with task identifiers, selected model, owner, and request payload.
- Keep source media URLs reachable long enough for APIDot to fetch them.
- Be explicit about subject, composition, output constraints, quality tier, and source media.
- Do not log prompts or source media URLs if they may contain private customer data.
